# Excel Financial Performance Dashboard 

## Overview
Interactive financial dashboard built in Microsoft Excel analyzing 
annual revenue, cost, and profit performance across 4 business 
divisions (North, South, East, West) for the full year 2023.

## Key Business Insights
- Total Revenue: ₹58,68,000 across all divisions
- Total Cost: ₹41,25,000
- Total Profit: ₹17,43,000
- Average Profit Margin: 29.40%
- North division consistently highest revenue every month
- Revenue grew steadily from Jan (₹4,13,000) to Dec (₹5,71,000)

## Dashboard Features
- 4 KPI Cards — Total Revenue, Total Cost, Total Profit, Avg Profit Margin %
- Monthly Financial Trend — line chart (Revenue, Cost, Profit)
- Profit by Division — bar chart comparing all 4 divisions
- Revenue vs Cost by Month — clustered column chart
- Profit Margin % by Month — line chart showing margin trend

## Excel Skills Demonstrated
- Pivot Tables and Pivot Charts
- Formulas: SUM, AVERAGE, MATCH
- Conditional formatting
- Data formatting and number formatting
- Dashboard design and layout
- Named tables and structured references

## Tools Used
Microsoft Excel (Advanced), Pivot Tables, Pivot Charts

## Dataset
Custom financial dataset — 48 rows of monthly divisional 
financial data across 12 months and 4 business divisions
